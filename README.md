# Horkan_etal26_NvecGenomes
Code used for assembly and analysis of N. vectensis genomes in Horkan et al 2026

This directory contains the following: 

- Library demultiplexing and initial read QC
- Primary, haplotype-aware, genome assembly with Hifiasm
- QC
- Scaffolding of primary contigs
- Cross reference backfilling
- Backfilling from current reference assemblies
- QC (including Cnidarian BUSCO)
- Gene model prediction
- Gene model validation with RNA seq across conditions and development
- Manual annotation cleanup
